Javascript runs in the browser, Case sensitive language. Interpreted Language, Does not need compilation. Runs line by line. Main Uses: Form validation - checking the forms for errors Pop up ads - usually opens automatically depending on user action Dynamic pages - (Static pages meaning content is fixed) User can modify the content of the site at runtime.
